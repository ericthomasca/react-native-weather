## Introduction

**React Native Weather** is a mobile weather app that allows users to check the current weather conditions, including temperature, humidity, wind speed, and more, for any location worldwide. It offers a user-friendly interface and supports both light and dark themes for a personalized user experience.

## Features

- Real-time weather data using the OpenWeather API
- Light and dark theme support
- Search for weather information for any location
- Detailed weather information, including temperature, humidity, and wind speed

## Getting Started

Follow the instructions below to set up and run the **React Native Weather** app on your local machine.

### Prerequisites

Before you begin, ensure you have the following prerequisites installed:

- Node.js (v18 or higher)
- npm or yarn
- Expo CLI

### Installation

Clone the repository and install the dependencies:

```bash
# Clone the repository
git clone https://github.com/yourusername/react-native-weather.git
```

### Navigate to the project folder

```bash
cd react-native-weather
```

### Install dependencies

```bash
npx expo install
```

### Usage

To use the app, follow these steps:

- Run the app using the following command:

```bash
expo start
```

- Open the Expo Go app on your mobile device or use an emulator to scan the QR code displayed in the terminal.

- Navigate through the app's screens:
  - Home: Displays detailed weather information for the selected location.
  - Location: Allows you to choose your desired location.
  - Settings: Allows you to toggle between light and dark themes.

If you find any issues or want to suggest improvements, please open an issue on the GitHub repository.
